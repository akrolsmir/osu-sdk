    osu!sdk

    Come help us make osu! more awesome! This is the start of an osu! client-side SDK framework which will allow for cool things to happen. For the time being, focus is being placed on editor extension via the AiMod window.

    If you have any specific queries, shout at @peppy. To get more added to the API, feel free to file issues on GitHub and we will review asap :).
